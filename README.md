# BMTest

Test project for BM company

## Launch

### Docker

Go to `src/` dirrectory.

Run this:
```
docker-compose up
```

And open this http://localhost:8080 in browser.


### Manual

 * Setup MS SQL Database.
 * Configure `ConnectionStrings.DefaultConnection` in `appsettings.json`
 * Compile and run
